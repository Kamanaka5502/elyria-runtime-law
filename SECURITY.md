This repository is a protected public proof surface. It is not a production deployment package.

## Public review scope

Security review is limited to public-safe materials:

```text
README.md
LICENSE
PROTECTED_SCOPE.md
docs/PRODUCTION_ENFORCEMENT_FABRIC.md
public category doctrine
public runtime-law vocabulary
public consequence-admission framing
```

## Do not publish here

```text
secrets or credentials
customer-specific material
client corridors
private runtime law
protected enforcement internals
private evaluator algorithms
production adapters
private schemas
receipt-generation internals
replay-verification internals
protected runtime substrate
production endpoint details
```

## Public issue scope

Public issues should be limited to:

```text
documentation clarity
public category wording
protected-scope boundary clarity
unsafe interpretation risk
broken public links
```

## Boundary

The absence of private runtime law, private evaluator logic, private schemas, client corridors, and production adapters is intentional.
